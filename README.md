# WaveNest

A decentralized music discovery platform with AI-generated recommendations built on the Stacks blockchain.

## Features
- Artist profile registration and management
- Song/Album NFT minting
- Listener profile and preferences tracking
- AI-powered music recommendations
- Token-based rewards system for active users

## Contract Functions
- Artist registration and profile management
- Song/Album NFT minting and management 
- Listener interaction tracking
- Recommendation engine integration
- Token rewards distribution

## Getting Started
1. Clone the repository
2. Install dependencies
3. Run tests using Clarinet
4. Deploy contract to testnet/mainnet
